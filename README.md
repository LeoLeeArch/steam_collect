# SteamPulse（演示版）

SteamPulse 是一个给普通玩家用的 iPhone 小工具：
**快速看 Steam 热门游戏现在多少钱、是不是接近历史低价。**

> 适合“先看值不值得买，再决定要不要点进 Steam”的场景。

---

## 这个 App 现在能做什么

1. 看热门游戏列表（热卖 / 热玩）。
2. 看单个游戏详情：当前价、原价、折扣、历史最低价。
3. 给出“接近史低”提示，并在详情页说明判定依据。
4. 在设置页调整抓取参数并保存（本地保存）。

---

## 第一次打开，怎么用（30 秒）

1. 打开 **首页**：先看“首次使用说明”和“标签说明”。
2. 点右上角 **刷新**：拉取最新数据。
3. 切到 **热门榜单**：浏览列表。
4. 点任意游戏：进入 **详情页** 看价格依据。
5. 去 **设置**：调整参数并保存。
6. 回首页再点 **刷新**：按新设置拉取。

## 你会在界面里看到的关键标签

- **免费候选**：免费领取候选位（当前版本仍为占位说明）。
- **历史最低**：当前价等于历史最低价。
- **接近史低**：当前价小于等于历史最低价上浮 3%。

## 本地运行（第一次接触也能照做）

1. 在 macOS 上安装 Xcode（建议 15+）。  
2. 打开 `SteamPulse.xcodeproj`。  
3. 选择一个 iOS 17+ 模拟器（例如 iPhone 15）。  
4. 按 `⌘R` 运行。  

---

## 如何验证主要功能（建议按这个顺序）

1. **首页**：确认看到“首次使用说明 / 标签说明 / 热门游戏”。  
2. **刷新**：点击右上角“刷新”，观察列表更新与“上次刷新”时间。  
3. **榜单**：切到“热门榜单”，确认每项可见当前价/原价/折扣/历史最低。  
4. **详情**：点击任意游戏，确认“判定依据”“规则状态”可见。  
5. **设置**：修改参数并保存，再回首页或榜单手动刷新验证生效。  
6. **诊断**：查看“最近刷新”与“缓存状态”是否更新。  

---

## 无网络时怎么演示（现场兜底）

- 先说明：当前版本是“本地优先 + 手动刷新”的监控工具。  
- 若刷新失败：展示错误态与诊断页“最近刷新状态”，说明已做失败回退。  
- 若有历史缓存：可继续展示已有列表和详情，解释这是本地缓存能力。  
- 若无缓存：展示空态并说明“这是首次运行 + 当前网络不可用”的预期表现。

## 隐私与数据说明（非技术版）

- 这是本地优先应用：配置与历史数据保存在手机本地文件。
- 当前版本不需要账号系统，不做云同步。
